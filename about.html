ICD11 DISEASE ARTICLE DATABASE

  • About

  • This website provides access to database containing links to Wikipedia articles assigned to specific diseases or their respective ICD11 codes. The database is hosted on a MongoDB server which is querried by REST API written in Java using Spring. Here you can also download the latest version of database in Protege .owl file format which is generated by a parser built into the API. This project was created in 2019 at Poznan University of Techonogly, Faculty of Electrical Engineering.
